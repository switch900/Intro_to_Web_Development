Chapter 3 Web Research part 1

Website URL Description
W3 CSS Home Page https://www.w3.org/Style/CSS/ W3C Home Official Homepage for CSS. Contains tutorials, links to software and news about CSS.
Mozilla CSS Development Docs https://developer.mozilla.org/en-US/docs/web/css Mozilla gives information for developers about CSS. With tutorials, news and information developers will need to know when using CSS with Firefox.
Sitepoint CSS Tutorials https://www.sitepoint.com/html-css/css/ Tutorials, books and news about CSS.
CSS-Tricks https://css-tricks.com/ Tutorials, news and a really useful reference guide to the many features of CSS. Also has code snippets that might be useful.
Noupe Design Blog https://www.noupe.com/category/design A blog showcasing some great website design ideas using CSS.
W3 CSS 2.1 Specifications https://www.w3.org/TR/CSS2/ W3 CSS 2.1 Specifications. A dry read but literally the users guide for CSS 2.1.
CSS Discuss http://www.css-discuss.org/ A place for authors to discuss real-world uses of CSS.
A List Apart CSS Discussion http://alistapart.com/topic/code/css/ The latest news about CSS and web design.
CSS Zen Garden http://www.csszengarden.com/ A plethora of CSS design templates.
Added Bytes CSS Cheat Sheet https://www.cheatography.com/davechild/cheat-sheets/css2/ A quick reference guide for CSS, listing selector syntax, properties, units and other useful bits of information.
Blueprint CSS http://www.blueprintcss.org/ Blueprint is a CSS framework, which aims to cut down on your development time. It gives you a solid foundation to build your project on top of.
